Сделай первый шаг в освоении акварельной живописи!

Сайт посвящен обзору обучения рисования акварелью на Youtube. Пошаговое повторение за художниками научит правильно выражать цвета с помощью акварельных красок и добиваться нужного результата особыми техниками рисования.


Для работы понадобятся следующие материалы:


  • набор художественной акварели,
  • мягкие круглые кисти разного размера,
  • бумага для рисования акварелью целлюлозная, плотностью не менее 200 г/м2,
  • бумажный скотч для фиксации листа,
  • вода,
  • палитра,
  • линер или мягкий карандаш.
